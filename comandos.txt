-Front-End

1) Inicio

yarn create react-app web --template typescript

2) Iniciar o projeto

yarn start

3) Rotas

yarn add react-router-dom
yarn add @types/react-router-dom -D

-Back-End

1) yarn add typescript -D

2) criar arquivo de configuração typescript

yarn tsc --init

3) Usada para executar os typescript

yarn add ts-node-dev -D

4) Express

yarn add express
yarn add @types/express -D

5) Banco de Dados

yarn add knex sqlite3

6) Comando para executar migration

yarn knex:migrate

7) Adição dos cors

yarn add cors
yarn add @types/cors -D
